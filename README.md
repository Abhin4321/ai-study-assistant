# ai-study-assistant


AI Study Assistant (LLM-Powered Learning Tool)
Overview

The AI Study Assistant is a web-based application that helps students transform lecture notes into structured study materials using Large Language Models (LLMs).

Users can paste raw lecture content, and the system uses AI to automatically generate:

concise summaries
key concepts
study questions (quizzes)
flashcards
context-aware Q&A responses

This project demonstrates how modern AI applications are built by integrating a backend server with OpenAI’s API to process and transform unstructured text into useful learning resources.

Problem It Solves

Students often struggle with:

large amounts of unorganized lecture notes
inefficient study methods
lack of practice questions
difficulty identifying key concepts

This tool helps convert raw notes into structured, study-ready content instantly, improving learning efficiency.

Features : 
Text Input System
Paste lecture notes directly into the application
No file uploads required (simple and fast workflow)
AI Summary Generator
Produces concise summaries of lecture content
Extracts key ideas and important definitions
Quiz Generator
Generates multiple-choice questions (5–10)
Provides correct answers for self-assessment
Flashcard Generator
Converts notes into Q/A flashcards
Optimized for quick review and memorization
Ask Your Notes (AI Chat Mode)
Users can ask questions like “Explain X” and AI responds using only the provided lecture context
